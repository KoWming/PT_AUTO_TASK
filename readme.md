# PT_AUTO_TASK
### 介绍
    这是一个为PT站点设计的自动化任务工具，可以用来执行一些站点的每日任务，包括但不限于签到、喊话、领取任务等
### 使用方法
推荐使用青龙脚本定期执行，也可以使用其他方式，如Github Actions、Docker等
